  ## HTML basic

- HTML : hyper text markup language
  - hyper text :하이퍼 링크로 연결된 문서 => 웹페이지(컨텐츠 , 구조)
  -markup : 표시
  -language : 언어  

- HTML 문법
  - 명칭 : Tag / Element
  - 구성 : 시작태그 ~ 종료태그
  - 종료태그가 없는 태그 : 빈태그 (Empty Element)

```
<tag> content </tag> : Element

<tag...> Empty Element

```
